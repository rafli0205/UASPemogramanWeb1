# Book Manager (UAS Pemrograman Web 1)

## Deskripsi
Aplikasi web sederhana untuk manajemen data buku (CRUD) berbasis PHP.
Fitur utama: login, kelola data buku (tambah, lihat, edit, hapus), dan logout.

## Teknologi yang digunakan
- PHP (Native/OOP)
- MySQL (via XAMPP)
- HTML/CSS (dan Bootstrap jika dipakai)
- Composer (jika project memakai autoload/dependency)

## Fitur Aplikasi
- Autentikasi:
  - Login
  - Logout
- Manajemen Buku:
  - Menampilkan daftar buku
  - Tambah buku
  - Edit buku
  - Hapus buku

## Struktur Folder (ringkas)
- `app/` : logic aplikasi (Controller, Model, dll)
- `public/` : halaman yang diakses user (entry point, asset)
- `vendor/` : dependency composer (jika tidak di-push, generate ulang via composer)
